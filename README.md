# 居民低碳行为数据分析与智能减碳推荐系统

## 项目简介

本系统是一个基于大语言模型的居民低碳行为数据分析与智能减碳推荐系统，旨在帮助居民了解自己的碳排放情况，获取个性化的减碳建议。

## 功能特点

### 1. 数据层
- 支持CSV导入，包含三类数据：
  - 垃圾分类数据（日期、投放类型、是否正确分类、频次）
  - 用电数据（日期、用电量、用电时段、用电设备）
  - 出行数据（日期、出行方式、距离、能源类型等）
- 内置模拟测试数据，可直接运行演示

### 2. 统计分析
- 自动清洗数据，处理缺失值和异常值
- 统计用电量、出行结构、垃圾分类正确率
- 计算个人碳排放量
- 使用Plotly图表可视化展示

### 3. 用户画像
- 三维打分体系（0-100分）：垃圾分类、用电习惯、出行方式
- 综合评分和等级评定（低碳达人/践行者/学习者/新手）
- 自动识别碳排放短板并给出改进建议

### 4. 大语言模型集成
- 个性化减碳建议：根据用户画像生成拟人化建议
- AI智能问答：支持自然语言提问低碳相关问题
- 自动生成个人低碳生活分析报告

## 技术栈

- **框架**: Streamlit
- **数据分析**: Pandas, NumPy
- **可视化**: Plotly
- **大语言模型**: Qwen API (阿里云通义千问)
- **配置管理**: python-dotenv

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置LLM API

在 `.env` 文件中配置Qwen API：

```env
OPENAI_API_KEY=your_qwen_api_key
OPENAI_BASE_URL=https://api.qwenlm.com/v1
LLM_MODEL=qwen-plus
```

### 3. 启动应用

```bash
streamlit run app.py
```

### 4. 登录系统

使用测试账户登录：
- user1 / password1
- user2 / password2
- admin / admin123

## 使用流程

1. **登录系统** → 输入用户名和密码
2. **加载数据** → 使用默认测试数据或上传自定义CSV
3. **查看统计图表** → 查看碳排放概览、垃圾分类、用电、出行统计
4. **查看用户画像** → 查看三维打分和碳排放短板
5. **获取减碳建议** → 由LLM生成个性化减碳方案
6. **AI问答咨询** → 用自然语言提问低碳相关问题

## 配置说明

### 碳排放系数配置 (config.py)

```python
CARBON_FACTORS = {
    "electricity": 0.581,  # 每度电碳排放系数 (kg CO2)
    "gasoline": 2.32,      # 每升汽油碳排放系数 (kg CO2)
    "diesel": 2.63,        # 每升柴油碳排放系数 (kg CO2)
    "natural_gas": 2.16,   # 每立方米天然气碳排放系数 (kg CO2)
}
```

### 评分权重配置 (config.py)

```python
SCORE_WEIGHTS = {
    "garbage": 0.3,    # 垃圾分类权重
    "electricity": 0.4, # 用电权重
    "transport": 0.3,  # 出行权重
}
```

## 数据格式

### 垃圾分类数据 (CSV)

| 日期       | 投放类型 | 是否正确分类 | 频次 |
|------------|----------|--------------|------|
| 2024-01-01 | 厨余垃圾 | 是           | 1    |
| 2024-01-01 | 可回收物 | 否           | 1    |

### 用电数据 (CSV)

| 日期       | 用电量(kWh) | 用电时段 | 用电设备 |
|------------|-------------|----------|----------|
| 2024-01-01 | 12.5        | 白天     | 空调     |

### 出行数据 (CSV)

| 日期       | 出行方式 | 出行距离(km) | 能源类型 | 油耗/电量(L/kWh) | 出行目的 |
|------------|----------|--------------|----------|------------------|----------|
| 2024-01-01 | 私家车   | 15.2         | 汽油     | 1.22             | 上班     |

## 注意事项

1. 如果未配置LLM API，系统会使用内置的模拟回复功能
2. 建议使用Python 3.9+版本运行
3. 首次运行会自动生成测试数据
